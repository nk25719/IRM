# Data Management User Guide

Open `Administration -> Data Management`.

Use `Download Template` before preparing data. Templates contain approved headers, instructions, accepted values, and example rows.

Use `Import Data` to upload `.csv` or `.xlsx` files. The system maps columns, previews rows, validates required fields, and stages results for review. Rows with blocking errors cannot be confirmed.

Use `Export Data` to select an implemented dataset, choose fields, preview rows, and download `.xlsx` or `.csv`.

Use `Import History` to review staged batches and download error reports.

Use `Validation Center` to review open and resolved validation issues. Resolution is read-only in this milestone because correction storage and resolver audit fields are not complete enough for safe edits.

Use `Data Health` to review real quality metrics such as unresolved validation errors, missing serial numbers, and unlinked equipment models. Unimplemented checks are marked unavailable.
